# Web app: Next.js 15 (App Router) + Tailwind CSS v4

This is a **Next.js web app** (App Router, React 19, TypeScript strict) styled with **Tailwind v4**
(config-less; theme tokens in `src/app/globals.css` via `@theme`). It was previously an Expo/React
Native app — that code is gone; do not reintroduce React Native, Expo, or native modules.

Before writing framework code, consult the current Next.js (App Router) and Tailwind v4 docs rather
than relying on memory of older versions or the prior RN stack.
